### 검색 기능
<br>
**공공데이터(https://www.data.go.kr/)에서 약 53000개의 약 정보와 100개의 약 부작용 정보를 파싱하여 데이터베이스에 저장함. <br>
Jquery의 Autocomplete 기능을 이용하여 자동완성 기능을 추가함.**
